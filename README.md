# SAT Pro Ultimate v40 (GitHub Pages)

## Deploy
1. Upload all files in this folder to your GitHub repo root.
2. Settings → Pages → Deploy from branch → main / root.
3. Open: https://<username>.github.io/<repo>/

## Notes
- Designed for online use (GitHub Pages). 
- Vocab loads from ./assets/vocab/sat5000.v1.json
- A Service Worker is included for offline caching.
